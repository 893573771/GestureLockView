# GestureLockView
### 手势解锁
修改[鸿洋的](http://blog.csdn.net/lmj623565791/article/details/36236113)手势解锁，继承RelativeLayout，可以自由定制手势点的样式的手势解锁
### 特性
- 实现ILockView，自由定制样式
- 支持修改/设置密码模式和验证密码模式
